## Buy now pay later application


### Project setup

1. Create a virtual env
    ```bash
    python3 -m venv venv
    ```

2. Activate the virtual env

    ```bash
    source venv/bin/active
    ```

3. Install the dependencies

    ```bash
    pip install -r requirement.txt
    ```

4. Run the project

    ```bash
    uvicorn mai:app --reload
    ```
